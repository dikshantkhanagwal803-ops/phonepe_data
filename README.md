PhonePe Data Analysis Dashboard (Tableau + SQL)

Project Overview
This project analyzes digital payment trends using PhonePe transaction data.  
The dashboard provides insights into transaction growth, state-wise performance, and payment behavior.


Key Insights
- Transactions grew exponentially after 2020
- Merchant payments dominate overall usage
- Telangana & Maharashtra lead in transaction value
- Strong upward trend indicates rapid digital adoption in India

Dashboard Features
- KPI Metrics (Total Transactions, Total Value, Avg Transaction)
- Transactions Over Time (Trend Analysis)
- State-wise Transaction Map (India-focused)
- Transaction Type Analysis
- Top States by Transaction Value
- Monthly Trend Analysis

Tools Used
- Tableau (Data Visualization)
- SQL (Data Analysis)
- Excel / CSV (Data Cleaning)


Files Included
- `phonepe_dashboard.twbx` → Tableau dashboard
- `phonepay_data.csv` → Dataset
- `analysis_queries.sql` → SQL queries
- `images/` → Dashboard screenshots
